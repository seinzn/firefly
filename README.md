# 掌心萤火 - 隐私政策

*最后更新：2026 年 2 月 10 日*

## 概述

掌心萤火（以下简称"本应用"）是一款专注陪伴工具。我们非常重视您的隐私，本政策说明我们如何处理您的信息。

## 信息收集

本应用**不收集**任何个人信息。具体而言：

- 不收集姓名、邮箱、电话号码等个人身份信息
- 不收集位置信息
- 不收集设备标识符用于追踪
- 不使用第三方分析或广告 SDK
- 不存在用户账户或登录系统

## 本地数据

本应用在您的设备本地存储以下数据，用于提供核心功能：

- **专注记录**：专注时长、完成时间等统计数据
- **解锁进度**：星座图鉴的解锁状态
- **用户设置**：您的偏好设置（如默认时长、显示选项）

这些数据仅存储在您的设备上，不会上传至任何服务器。

## iCloud 同步

如果您启用了 iCloud，上述本地数据可能通过 Apple 的 iCloud 服务在您的设备间同步。该同步完全由 Apple 管理，受 [Apple 隐私政策](https://www.apple.com/legal/privacy/) 保护。

## 订阅与购买

本应用提供应用内订阅服务，所有交易通过 Apple 的 App Store 处理。我们不会接触您的支付信息。相关政策请参阅 [Apple 隐私政策](https://www.apple.com/legal/privacy/)。

## 第三方服务

本应用不使用任何第三方服务、SDK 或分析工具。

## 儿童隐私

本应用不针对 13 岁以下儿童，也不会有意收集儿童的任何信息。

## 隐私政策变更

如本政策发生变更，我们将在此页面更新内容并修改上方日期。

如对本隐私政策有任何疑问，请通过以下方式联系：

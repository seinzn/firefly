## 常见问题

**Q：专注中途退出 App 怎么办？**
短时间内返回可以继续，专注记录不会丢失。

**Q：如何恢复已购买的订阅？**
打开 App → 订阅页面 → 点击「恢复购买」。

**Q：数据会同步到其他设备吗？**
订阅用户开启 iCloud 后，专注记录和图鉴进度会自动同步。

**Q：如何取消订阅？**
打开 iPhone「设置」→ 点击你的 Apple ID → 订阅 → 掌心萤火 → 取消订阅。
